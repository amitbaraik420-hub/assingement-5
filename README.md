1️. var, let, const difference

var: A variable declaration  with function scope that allows re-declaration and reassignment.

let: A variable declaration keyword with block scope that s reassignment but not re-declaration in  same scope.

const: A variable declaration keyword with block scope  cannot be reassigned or re-declared after initialization.

2️. Spread Operator (...)

The spread operator is a JavaScript syntax used to expand or unpack elements of an array or properties of an object into another array, object, or function call.

3️.map(), filter(), forEach() difference

map: A method that creates a new array by applying a function to each element of the original array.

filter: A method that creates a new array containing  the elements that pass a specified condition.

forEach: A method that executes a function for each  of an array without returning a new array.

4️. Arrow Function

An arrow function is a shorter syntax for writing functions in JavaScript, introduced in ES6, which does not have its own this context.

5️. Template Literals

Template literals are string literals in JavaScript that allow embedded expressions, multi-line strings, and string interpolation using  `.
